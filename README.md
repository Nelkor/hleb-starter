# HLEB Starter

A starter project using [microframework HLEB](https://github.com/phphleb/hleb)

## Installation and launch

1. Downloading the required images:  
  `docker-compose pull`
2. Building our own image:  
  `docker-compose build`
3. Starting services  
  `docker-compose up -d`

## Shutting down

- Pausing containers  
  `docker-compose stop`
- Removing containers  
  `docker-compose down`
